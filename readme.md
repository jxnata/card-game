# Regras do jogo

## 1. Objetivo

Ter a menor soma possível nas 3 cartas ao final da rodada.
O jogador que chegar a 100 pontos é eliminado do jogo.

## 2. Preparação

-   Cada jogador recebe 3 cartas, colocadas viradas para baixo sobre a mesa, à sua frente.
-   No início da rodada, cada jogador pode olhar apenas 2 das suas 3 cartas.
-   Após alguns segundos, as cartas devem ser viradas para baixo novamente.
-   A terceira carta não pode ser vista nesse momento.
-   O restante do baralho forma o monte de compras.

## 3. Valores das cartas

-   Cartas numéricas valem seu valor normal (2-10).
-   J (Valete) vale 0.
-   Q (Dama) vale 11.
-   K (Rei) vale 12.
-   A (Ás) vale 1.

## 4. Turno do jogador

No seu turno, o jogador pode:

-   Comprar 1 carta do monte, ou

-   Comprar a última carta do descarte.

Depois disso, pode:

-   Trocar a carta comprada por qualquer uma das suas 3 cartas, ou

-   Descartar a carta comprada e manter suas cartas.

## 5. Memória e dedução

O jogador pode usar a memória para lembrar das cartas que viu no início.
Não é permitido olhar cartas viradas para baixo fora do momento inicial ou da revelação final.
O jogador só tem como saber a carta do meio (a que ele não viu no início) trocando ela.

## 6. Pedido de parada

A qualquer momento no próprio turno, após comprar e decidir trocar ou não, o jogador pode pedir para parar o jogo.
Ao pedir para parar, a rodada entra na fase final.

## 7. Revelação das cartas

-   Todos os jogadores revelam suas 3 cartas.
-   Cada jogador soma o valor das suas cartas.

## 8. Condição de vitória

**O jogador que pediu para parar:**

-   **Vence** a rodada se sua soma for estritamente menor que a soma de todos os outros jogadores.
-   **Perde** a rodada se qualquer outro jogador tiver soma igual ou menor que a dele.

## 9. Pontuação

Se o jogador pede para parar e _vence_:

-   Ele não soma pontos.
-   Todos os outros jogadores somam os pontos das suas próprias cartas.

Se o jogador pede para parar e _perde_:

-   Ele soma todos os pontos de todas as cartas de todos os jogadores.
-   Os outros jogadores não somam pontos.

## 10. Fim automático da rodada

Se o monte de compras ficar vazio durante uma rodada, a rodada termina automaticamente.
Neste caso, todos os jogadores revelam suas cartas e cada um soma os pontos das suas próprias cartas.
Esta é uma condição de fim diferente do pedido de parada.

## 11. Eliminação

Jogadores acumulam pontos ao longo das rodadas.
Ao atingir 100 pontos, o jogador é eliminado do jogo.
O jogo continua até restar apenas um jogador não eliminado, que no final é o vencedor!
